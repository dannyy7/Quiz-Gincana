## Sobre o projeto

O **Quiz Gincana** é uma aplicação web desenvolvida em **React.js** com integração ao **firebase**, criada para o entreterimento de pessoas por meio de um jogo

O sistema permite:
- Criar o próprio **usuário**
- Criar e gerenciar **quizes**
- Criar e entrar em lobbys **online**

> O projeto foi criado visando a diversão de seus usuários

---

## Tecnologias utilizadas

- **React.js (vite)** - Iterface dinâmica e responsiva
- **Firebase** - Banco de dados e hospedagem
- **CSS** - estilização

## Como executar localmente

### 1. Clonar repositorio

```bash
git clone https://github.com/dannyy7/Quiz-Gincana
cd quizgincana
```

### 2. Instalar dependências

Ao colocar os códigos abaixo no terminal as dependências do react, firebase e do router serão instaladas.

```bash
npm install
npm install firebase
npm install react-router-dom
```


### 3. Criar o aquivo .env

```.env


```

### 4. Rodar o projeto

Ao enviar o código abaixo no terminal o aplicativo estará disponível no link fornecido

```bash
npm run dev
```
