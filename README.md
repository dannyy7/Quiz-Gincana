## Sobre o projeto

O **Quiz Gincana** é uma aplicação web desenvolvida em **React.js** com integração ao **firebase**, criada para o entreterimento de pessoas por meio de um jogo

O sistema permite:
- Criar o próprio **usuário**
- Criar e gerenciar **quizes**
- Criar e entrar em lobbys **online**

> O projeto foi criado visando a diversão de seus usuários

---

## Tecnologias utilizadas

- **React.js (vite)** - Iterface dinâmica e responsiva
- **Firebase** - Banco de dados e hospedagem
- **CSS** - estilização

## Como executar localmente

### 1. Clonar repositorio

```bash
git clone https://github.com/dannyy7/Quiz-Gincana
cd quizgincana
```

### 2. Instalar dependências

Ao colocar os códigos abaixo no terminal as dependências do react, firebase e do router serão instaladas.

```bash
npm install
npm install firebase
npm install react-router-dom
```


### 3. Rodar o projeto

Ao enviar o código abaixo no terminal o aplicativo estará disponível no link fornecido

```bash
npm run dev
```

## Funcionalidade extra

### Customização de Avatar

O usuário pode selecionar um personagem desenhado à mão, utilizado como sua foto de perfil nas salas e rankings.

### Salvamento automático (Autosave)

Durante a criação ou edição de perguntas, todas as alterações são salvas automaticamente, evitando perda de progresso.

### Pódio e ranking final dinâmico

Ao término da partida, a página exibe um pódio estilizado mantendo a estética ilustrada do projeto.
Caso nem todas as posições sejam ocupadas, placeholders aparecem de forma coerente com o design.
Além disso, enquanto as informações da página carregam, o design é mantido por uma ‘versão teste’, proporcionando uma experiência melhor ao preservar o estilo personalizado e evitar telas de carregamento.

### Sistema de salas online

O usuário pode criar salas, entrar através de códigos e participar de partidas em tempo real.

### Interface coerente e imersiva

A maioria das telas do projeto possuem um design design próprio que incluem desenhos feitos no celular por meio do aplicativo ibisPaint X, mantendo o estilo visual exclusivo do projeto.

### Motivo da implementação

As funcionalidades extras foram adicionadas para tornar o Quiz Gincana mais do que um simples jogo, a intenção das implementações é criar uma experiência divertida, personalizada e visualmente marcante.
Entre os principais motivos:

### Melhor experiência do usuário

Funcionalidades como autosave, ranking e avatar próprio deixam o uso mais fluido e intuitivo.

### Identidade visual única

Como muitas das imagens foram cuidadosamente desenhadas, era essencial que cada nova função respeitasse essa estética, reforçando o estilo próprio do projeto.

### Maior engajamento

A personalização visual, o pódio final e as salas interativas incentivam o usuário a explorar mais o aplicativo.

### Fortalecer o aspecto social

O ranking, os personagens personalizados e os lobbys tornam a experiência mais competitiva e colaborativa.

### Segurança e confiabilidade

O autosave e as melhorias de edição evitam perda de dados, trazendo mais estabilidade ao sistema.
